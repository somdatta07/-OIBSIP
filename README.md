# 🌸 Iris Flower Classification using Machine Learning

## 📌 Project Overview

The Iris Flower Classification project is a Machine Learning application that predicts the species of an Iris flower based on its physical measurements. The model is trained using the famous Iris dataset and can classify flowers into one of three species:

- Iris Setosa
- Iris Versicolor
- Iris Virginica

This project demonstrates the complete Machine Learning workflow, including data loading, preprocessing, model training, evaluation, and prediction.
Project Explanation
Dataset

The Iris dataset contains:

150 flower samples
4 features for each sample
3 flower species
Steps Performed
Load the Iris dataset.
Split data into training and testing sets.
Train a Random Forest classifier.
Evaluate the model using accuracy and classification report.
Predict the species of a new flower.
---

## 🎯 Objectives

- Understand the basics of supervised machine learning.
- Train a classification model using the Iris dataset.
- Evaluate model performance using accuracy and classification metrics.
- Predict flower species from new measurements.

---

## 📊 Dataset Information

The Iris dataset contains 150 samples of iris flowers with the following features:

| Feature | Description |
|----------|------------|
| Sepal Length | Length of the sepal (cm) |
| Sepal Width | Width of the sepal (cm) |
| Petal Length | Length of the petal (cm) |
| Petal Width | Width of the petal (cm) |

### Target Classes

| Class | Species |
|---------|----------|
| 0 | Iris Setosa |
| 1 | Iris Versicolor |
| 2 | Iris Virginica |
